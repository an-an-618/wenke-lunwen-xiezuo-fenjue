# 引文真实性与反幻觉审校

## 核验原则

文科论文最容易失守的地方不是文风，而是“看似有出处、实则无法对照”的引用。所有引用都必须通过四级核验：

1. **题录核验**：作者、标题、刊物、年份、卷期、页码是否真实。
2. **全文核验**：是否读到全文，而不是只读摘要或二手转述。
3. **观点核验**：正文归给某文献的观点，文献中是否真的出现。
4. **用途核验**：该文献是否适合支撑当前论点，还是被过度延伸。

## 禁止写法

- “有学者指出……”但没有文内引用。
- “某某认为……”但只在参考文献列出该文。
- 用文献标题推测观点。
- 把摘要中的一句话扩写成作者完整立场。
- 把多个学者的观点混合后归给其中一人。
- 引用二手文献时假装读过一手文献。

## 核验表字段

每条引用建议建立核验表：

- 正文句子：论文中实际写下的引用性表述。
- 文内引用：如 `(王爱松, 2008)`。
- 文献原句或摘要：不超过必要长度，保留页码或段落位置。
- 可支持程度：完全支持、部分支持、不支持、需要改写。
- 修订动作：保留、弱化、删去、改为自我判断、另找文献。

## APA 文内引用要点

- 转述观点：`（作者, 年份）`。
- 直接引用：`（作者, 年份, p. 页码）` 或中文论文常用 `（作者, 年份：页码）`，按用户指定格式统一。
- 多作者、同年多文、外文作者按 APA 规则处理。
- 参考文献列表必须与正文引用互相对应。

## 引用判定标准

可写“某某认为”的条件：

- 文献中明确提出该判断，或该判断是作者论证的直接结论。
- 不是从标题、关键词或摘要中猜测出来。
- 不是把该文献的研究对象误认为该文献的观点。

只能写“本文借助某某的讨论”的条件：

- 文献提供的是概念、问题意识、背景材料，而不是直接支持本文判断。

必须改成“本文认为”的条件：

- 判断来自自己的综合推理，文献只提供间接背景。

## 使用脚本

可先运行：

```bash
python scripts/citation_skeleton.py paper.docx --out citation-audit.md
```

然后逐条补入证据。脚本只能抽取候选引用，不能判断真实性。
