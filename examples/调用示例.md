# 调用示例

## 课程论文

```text
使用 $wenke-lunwen-xiezuo-fenjue，围绕《红楼梦》与某当代小说的互文关系写一篇 8000 字中文系课程论文。要求先做文献综述，再提出中心论点，使用 APA 文内引用，最后输出 Word 和引文核验报告。
```

## C 刊标准审稿

```text
使用 $wenke-lunwen-xiezuo-fenjue，扮演严苛中文系教授审查这篇论文。重点检查中心论点、结构、小标题、文本细读、文献真实性和引用格式。审稿后再扮演作者写修订方案，并另存修订稿，不要覆盖原稿。
```

## 引文专项核验

```text
使用 $wenke-lunwen-xiezuo-fenjue，检查这篇论文中所有文内引用是否真的出自 Zotero 里的文献。凡是没有证据的地方，标记为不支持，并提出改写方案。
```

## Word 格式体检

```text
使用 $wenke-lunwen-xiezuo-fenjue，对这个 DOCX 做格式审校：不得有页头，不得有图表，标题不得标蓝，参考文献和正文引用必须对应。
```
